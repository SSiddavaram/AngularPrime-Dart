The <pui-datatable> is an advanced datatable widget insprired by the datatable component of PrimeFaces.

The current version is an early prototype that only just works. Be prepared for API changes.

Planned changes:

  • Get rid of by moving the pui-repeat directive to .
  • Allow arbitrary content in the table cell (for instance, editable input fields).
  • Add filters.
  • Add paginators.
